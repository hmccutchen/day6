Starting a new project

1. create a project folder for your project

You can use the command mkdir (folder name) or you can do this through the interface.

2. Initilize git repository inside project

Navigate to project folder inside of terminal and run the following command: git init

You should see the following output: Initialized empty git repository in /users/(parent folder)/(project)/.git/

3. Open a browser and go to Github and create a new repo.

once you create a new repo, you will copy and paste the following commands into terminal. these commmands set the remote to point to your project folder ...

Git remote add origin https://github.com/project-name.git

As well as stsrting a new branch for you to work on. this branch should be called master

git push -u origin master

Youre all set

Now read about adding files to your project and pushing your code to Github!
